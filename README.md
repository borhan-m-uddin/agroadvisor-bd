# AgroAdvisor BD — Agricultural Disease Advisory Chatbot

Bilingual (English/Bengali) RAG chatbot for crop disease management in Bangladesh.

**Knowledge Base:** 32+ documents from BRRI, IRRI, FAO, BARI, USDA + structured knowledge files  
**Total Chunks:** ~35,000+ vector embeddings  
**Languages:** English and Bengali (বাংলা)  
**Built with:** ChromaDB, sentence-transformers, LangChain, Streamlit, Groq API

## Features
- Crop disease identification and management
- Fertilizer recommendations
- Pest management guidance
- Climate impact information
- Bilingual responses (English + Bengali)
